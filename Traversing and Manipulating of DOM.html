DOM Selection Methods

These methods allow you to select elements from the DOM tree.

1. getElementById()

Selects an element by its ID attribute.

document.getElementById('myElement');

2. getElementsByClassName()

Selects all elements with a specific class name.

document.getElementsByClassName('myClass');

3. getElementsByTagName()

Selects all elements with a specific tag name.

document.getElementsByTagName('div');

4. querySelector()

Selects the first element that matches a CSS selector.

document.querySelector('#myId .myClass');

5. querySelectorAll()

Selects all elements that match a CSS selector.

document.querySelectorAll('div.myClass');

DOM Traversal

Traversal means moving through the DOM tree to select elements relative to other elements.

1. Downwards Traversal

Moving from parent to child elements.

// Using querySelector/querySelectorAll
parent.querySelector('.child');

// Using children property
parent.children;

2. Upwards Traversal

Moving from child to parent elements.

// Using parentElement
child.parentElement;

3. Sideways Traversal

Moving between sibling elements.

// Using nextElementSibling and previousElementSibling
element.nextElementSibling;
element.previousElementSibling;

// Using parentElement + children + index
parent.children[index];

Practical Example: Dynamic List

This example combines selection, traversal, and manipulation to create an interactive list.

  • Item 1
  • Item 2
  • Item 3
function addListItem() {
  const input = document.getElementById('itemInput');
  const list = document.getElementById('itemList');
  if (input.value) {
    const newItem = document.createElement('li');
    newItem.textContent = input.value + ' ';
    const removeBtn = document.createElement('button');
    removeBtn.textContent = 'Remove';
    removeBtn.onclick = function() {
      this.parentElement.remove();
    };
    newItem.appendChild(removeBtn);
    list.appendChild(newItem);
    input.value = '';
  }
}
